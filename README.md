🛠 نصب و اجرا

برای اجرای پروژه به Node.js نیاز داری.

1️⃣ کلون پروژه

git clone https://github.com/alishabani/alishabani.resume.git
cd alishabani.resume


2️⃣ نصب پکیج‌ها

npm install


3️⃣ اجرای لوکال

npm run dev


4️⃣ ساخت نسخه Production

npm run build


5️⃣ دیپلوی روی GitHub Pages

npm run deploy
